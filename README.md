# Zufallszitategenerator

Ein einfacher Generator für zufällige Zitate auf Deutsch, erstellt mit **PHP** und **HTML/CSS**.  

Jedes Mal, wenn die Seite geladen wird oder man auf den Button klickt, wird ein neues Zitat angezeigt.

## Funktionen

- Zufällige Anzeige von Zitaten bei jedem Laden der Seite.
- Button „Noch ein Zitat“, um ein neues Zitat zu generieren.
- Einfache Struktur mit **PHP**, **HTML** und **CSS**.

## Installation / Nutzung

1. Repository auf deinen Rechner klonen oder herunterladen:

```bash
git clone https://github.com/dein-benutzername/zufallszitategenerator.git
```
2. Dateien auf einen Webserver kopieren, der PHP unterstützt (z.B. XAMPP, MAMP, oder einen echten Server).

3. Im Browser die Datei index.php öffnen.

4. Den Button „Noch ein Zitat“ klicken, um neue Zitate zu sehen.

## Struktur
```
zufallszitategenerator/
│
├─ index.php       # Hauptseite mit PHP-Code
├─ zitate.php      # Array mit Zitaten
├─ style.css       # CSS-Stile für die Seite
└─ README.md       # Beschreibung des Projekts
```
## Zitate bearbeiten / hinzufügen

- Öffne die Datei zitate.php.

- Füge neue Zitate im Array $zitate hinzu, z.B.:
```
$zitate[] = "Dein neues Zitat hier — Autor";
```
